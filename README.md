# 📘 Revisão de Python – Fundamentos (VS Code)

Este repositório contém uma revisão prática dos fundamentos da linguagem **Python**, baseada na aula  
**Curso Completo de Python 2026 (Iniciantes)** do canal **Hashtag Programação**.

Todo o conteúdo foi estudado a partir do início da aula (00:00) e testado diretamente no **Visual Studio Code (VS Code)**.

---

## 🧠 Conteúdos Revisados

### 1. Variáveis e Tipos de Dados
- Criação de variáveis
- Tipos básicos: `int`, `float`, `str`
- Uso do `print()`
- Tipagem dinâmica do Python

---

### 2. Strings (Textos)
- Concatenação de textos
- f-strings
- Métodos de string:
  - `lower()`, `upper()`, `capitalize()`, `title()`
  - `replace()`, `find()`
- Uso do `len()`

---

### 3. Entrada de Dados
- Uso do `input()`
- Conversão de tipos (`int`, `float`)
- Interação com o usuário no terminal

---

### 4. Estruturas Condicionais
- `if`, `elif` e `else`
- Operadores de comparação
- Importância da indentação
- Exemplos com metas, vendas e bônus

---

### 5. Listas
- Criação e acesso por índice
- Índices positivos e negativos
- Métodos:
  - `append()`, `remove()`, `sort()`
- Funções:
  - `len()`, `sum()`, `max()`, `min()`
- Uso do operador `in`

---

### 6. Dicionários
- Estrutura `chave : valor`
- Acesso, edição e remoção de dados
- Uso do `pop()`
- Verificação de chaves com `in`
- Criação de dicionário a partir de listas com `zip()`

---

### 7. Laços de Repetição
- `for item in lista`
- `for i in range()`
- Uso do `enumerate()`
- Percorrendo listas, dicionários e tuplas

---

### 8. Funções
- Criação de funções com `def`
- Parâmetros e retorno com `return`
- Funções aplicadas a cálculos reais
- Uso de funções com `for`
- Padronização de casas decimais com `round()`

---

### 9. Tuplas
- Estruturas imutáveis
- Criação de tuplas
- Unpacking (desempacotamento)
- Retorno múltiplo de funções
- Tuplas dentro de listas

---

## 🛠️ Ambiente Utilizado
- Python 3
- Visual Studio Code
- Execução via terminal integrado

---

## ✅ Objetivo do Projeto

Consolidar os fundamentos do Python através de exemplos práticos, reforçando o raciocínio lógico e preparando a base para estudos mais avançados e projetos reais.
